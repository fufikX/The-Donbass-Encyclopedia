# Структура проекта "Детская энциклопедия природы Донбасса"

```
project/
│
├── index.html            # Главная страница
├── quiz.html             # Викторина (уже готова)
├── flora.html            # Страница о флоре (будет создана)
│
├── css/
│   ├── main.css          # Основные стили
│   ├── quiz.css          # Стили для викторины
│   └── flora.css         # Стили для страницы флоры
│
├── js/
│   ├── main.js           # Основной JavaScript
│   └── quiz.js           # JavaScript для викторины
│
├── images/
│   ├── logo.png          # Логотип сайта
│   ├── flora/            # Изображения растений для страницы флоры
│   │   ├── tulipan-shrenka.jpg
│   │   ├── kovyl.jpg
│   │   ├── ...
│   │
│   └── backgrounds/      # Фоновые изображения для карточек и секций
│       ├── flora-bg.jpg
│       ├── fauna-bg.jpg
│       ├── ...
│
└── video/
    └── nature.mp4        # Видео для фона на главной странице
```
